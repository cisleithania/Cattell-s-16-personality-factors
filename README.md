# Программное обеспечение для автоматического построения многофакторной модели индивида на базе опросника Кеттелла.

Программа осуществляет полный цикл выполнения многофакторного тестирования по опроснику Кеттелла в полной модификации 16PF. 
По окончанию прохождения теста автоматически рассчитываются личностные факторы. 
Результаты выдаются в текстовом виде, а также в виде круговых диаграмм.
